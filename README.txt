支持的功能
	1、仅支持POST请求
	2、所有未知的请求和错误只会返回404
	3、html文本传输
	4、简单的动态网页sdw(simple dynamic webpage)
	   使用命令行传输参数

基础配置
	在头文件中使用宏进行基础配置
	ROOTPATH	网页文件位置
	SDWPATH		动态网页文件位置，必须在网页文件的目录下
	PORT		端口

注意事项
	1、不要试图测试此程序的稳定性，这是一个脆弱的程序
	2、如果一个客户端于服务器建立TCP连接，但什么都不做，
	   则服务器只会等待。在测试过程中，我发现，
	   有些客户端比如edge，会无视应答中的Connection属性，
	   如果服务端关闭TCP连接，edge会立即建立新的TCP连接。
	3、对于长度超过限制的路径和参数无法正常的处理，而且可能发生崩溃
	4、不会对路径和参数进行检查，所以不要尝试攻击这个程序
	5、虽然只支持GET请求，但是能够识别所有请求，可以轻松的进行扩展
	6、有两个示例页面，一个静态，一个动态
